# OSS-Asset-Manager
System ITAM (IT Asset Management) dedykowany dla środowisk biurowo-produkcyjnych. Aplikacja umożliwia ewidencję sprzętu IT (terminale, telefony, laptopy) z precyzyjnym przypisaniem do stref operacyjnych (np. Finishing, Despatch) oraz pełną kontrolą nad nazewnictwem i numeracją seryjną zasobów.
